# 👻 Ghost Prank Python App

A fun Python prank project that surprises users with a sudden ghost jump scare! 😈  
Perfect for harmless fun with friends.

---

## 🎯 Features
- Fullscreen background display
- Sudden ghost image popup
- Spooky sound effects 🔊
- Simple and lightweight Python script

---

## 🛠️ Built With
- Python 🐍
- Pygame 🎮

---

## 🚀 How It Works
1. The program starts in fullscreen mode
2. Plays background spooky audio
3. Waits for a few seconds ⏳
4. Suddenly shows a ghost image with a scary sound 👻

---

## 📦 Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/ghost-prank.git
cd ghost-prank
````

### 2. Install dependencies

```bash
pip install pygame
```

---

## ▶️ Run the Project

```bash
main.py
```

---

## 📁 Required Files

Make sure these files are in the same folder:

* `main.py`
* `ghost.jpg`
* `ghost sound.mp3`
* `ghost sound 1.mp3`
